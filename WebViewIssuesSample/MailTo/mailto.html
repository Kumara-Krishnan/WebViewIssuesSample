Mail To Uri Page

Please make WebViewIssuesSample app as the default app for mailto protocol in settings->Apps->Default Apps->Email page

The Uri class throws a UriFormatException if there's a comma between the recipients in the mailTo url.
Clicking on the last link will throw a UriFormatException.
But as per the mail to url scheme comma is a valid character between the recipients
I have already raised a ticket in both the UWP user voice and the .NET developer community forum

Works mailto:majordomo@example.com?body=subscribe%20bamboo-l
Works mailto:joe@example.com?cc=bob@example.com&body=hello
Works mailto:?to=joe@example.com&cc=bob@example.com&body=hello
Works mailto:infobot@example.com
Throws UriFormatException mailto:someone@example.com,someoneelse@example.com?to=infobot@example.com&cc=infobot@example.com&subject=hello
